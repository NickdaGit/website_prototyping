Closures in Python

Functions that remember where they came from—like a backpack you pack once and carry forever.

A closure is a function that "closes over" variables from its outer scope. Even after the outer function finishes, the inner one keeps access.

def outer():
    x = 10
    def inner():
        print(x)  # still sees x!
    return inner

f = outer()
f()  # → 10

Got it? It's magic without the wand.
